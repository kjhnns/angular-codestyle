UI Bootstrap - Documentation

Eine Sammlung von Directives für Angular. Zugrunde liegt das Design Framework Twitter Bootstrap.

UI Bootstrap bringt lediglich die Funktionalitäten von Bootstrap zu angular, dass heißt das man kein JavaScript mehr braucht, sonder nur die Directives.

Zusätzlich bedeutet dies aber auch, dass man die CSS Klassen und Stylistischen Elemente die nur auf CSS basieren aus Bootstrap trotzdem nutzen kann.

Folgende Elemente aus Bootstrap werden durch UI Bootstrap dynamisch:

  • Accordion
  • Alerts
  • Buttons
  • Carousel
  • Collapse
  • Datepicker
  • Dropdown
  • Modal
  • Pagination
  • Popover
  • Progressbar
  • Rating
  • Tabs
  • Timepicker
  • Tooltip
  • Typeahead
